# Review Checklist — Idea Catalog

Use this during peer review. Check only what you actually verified.

- [ ] Бизнес-логика совпадает с описанием в связанном Issue
- [ ] Автор обрабатывает ошибки (`ApiError`) и не пропускает 500 без причины
- [ ] Теги и статусы идей согласованы с текущими правилами продукта
- [ ] Тесты покрывают новые ветки и падают, если сломать изменение
- [ ] Документация/шаблоны обновлены, если поведение API поменялось
